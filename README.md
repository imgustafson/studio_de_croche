# 🧶 Studio de Crochê

> Aplicativo web para crocheteiras organizarem projetos, calcularem materiais e consultarem referências, tudo em um lugar só.

**🌐 Acesse online:** [imgustafson.github.io/crochet-calc](https://imgustafson.github.io/crochet-calc)

---

## ✨ Funcionalidades

### 📋 Meus Projetos
Registre seus projetos com nome, tipo, cores do fio, quantidade de novelos, status de andamento, receita completa e foto. Acompanhe o progresso de cada um com os status *Planejado*, *Em andamento* e *Concluído*.

### 📦 Meu Estoque
Cadastre os novelos que você já tem em casa — marca, cor, espessura e metragem — para nunca comprar repetido ou esquecer o que tem disponível.

### 📖 Glossário de Pontos
Referência completa dos principais pontos de crochê com abreviações e descrições. Você também pode adicionar seus próprios pontos personalizados e salvá-los para consultar sempre que precisar.

### 🪡 Tabela de Agulhas
Conversão entre milímetros, numeração US e UK, com fio indicado e uso comum para cada tamanho.

### ⏱️ Contador de Fileiras
Contador com um clique para usar enquanto crocheta. Suporte a alarme configurável a cada X fileiras e salvamento de sessões com nome.

### 🎨 Paleta de Cores
Monte combinações de cores para seus projetos usando um seletor visual. Dê nome a cada cor e salve paletas completas para consultar depois.

### 🧵 Calcular Lã
Calcule quantos novelos você precisa com base na largura, comprimento, tipo de ponto, espessura do fio e gauge.

### 📐 Calcular Pontos
Usando sua amostra, calcule exatamente quantos pontos montar e quantas fileiras fazer para o tamanho desejado.

### 📏 Converter Tamanhos
Redimensione receitas de PP até XGG com fator de conversão automático para pontos e novelos.

## 🛠️ Tecnologias

- **HTML5, CSS3 e JavaScript** puro — sem frameworks ou dependências externas
- **Google Fonts** — Playfair Display + DM Sans
- **localStorage** para persistência dos dados no navegador
- Arquivo único `index.html` — abre direto no navegador, sem instalação
